Perspectives

Perspectives can be best seen as kind of pages within the application which assemble Views and Editors into a concise UI or dash board. Perspectives can be used to setup concise workbenches with a strong focus on the current task.

Within Purifinity Client, it is possible to setup multiple different perspectives. These perspectives provide different information and due to they are on the same workspace and work on the same analysis projects, provide a rich variety of information in an organized and clean way.

Opening of an Installed Perspective

In Purifinity Client, a lot of perspectives are already build in. These perspectives can be accessed with Window > Show Perspective.... A dialog is show which provides a list of all already defined perspectives. These perspectives are installed along with plug-ins.

The Perspective is just selected and the selection is confirmed with OK to get it opened.

The same action can be performed with a button in the upper right corner of the application.

Closing of an Perspective

The currently opened perspective can be closed with Window > Close Perspective.

Switching Perspectives

Switching of a perspective can be done via menu File > Show Perspective..., too. As soon as the selected perspective is already opened, the perspective is shown, but not re-opened or reset.

Additionally, a button list on the upper, right corner of the application lists all open perspectives. A click on the button opens the selected perspective.

Creating Customized Perspectives

An open perspective or after all perspectives an empty perspective can be customized with Window > Customize Perspective... and by showing and closing parts. As soon as the perspective is a state which should be preserved, the perspective can be saved into the workspace by Window > Save Perspective As... under a custom name.

Custom perspectives can also be selected with Window > Select Perspective....
