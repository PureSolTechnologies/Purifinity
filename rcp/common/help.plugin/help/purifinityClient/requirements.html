Requirements

To run the Purifinity client the following requirements are needed:

Hardware

The following hardware is needed as minimum and recommendation.

Minimum

  1. 32bit or 64bit Intel compatible CPU
  2. min. 1GB RAM
  3. 10GB disk space
  4. Screen resolution of at least 1280x1024

Recommended

  1. 64bit Intel compatible multi-core CPU (Purifinity is multi-core optimized, the more cores are available the faster analyses and evaluations are performed).
  2. 4GB of RAM (the more cores are available the more RAM is needed due to analyses and evaluations which are performed in parallel)
  3. Plenty of disk space (>100GB) for trend charts
  4. Full HD screen resolution of 1920x1080 for comfortable charting and full usage of complex perspectives

Operating System and Software

  1. 32bit or 64bit version of Linux or Windows (only 64bit versions are tested, but 32bit versions should run without issues due to usage of Java)
  2. A suitable Java 7 for the used OS needs to be installed and available for the user who runs Purifinity

As soon as these requirements are met, you can proceed with the installation of the client.
