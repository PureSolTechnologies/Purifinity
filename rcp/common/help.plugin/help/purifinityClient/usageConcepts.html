Usage Concepts

The Purifinity Client is based on Eclipse RCP platform. All usage concepts and principle are found in Purifinity Client.

The Eclipse UI is very intuitive and self-explanatory. That's why in this user's manual only the basic concepts are explained to help to get users up to speed. If a more detailed manual is needed, please find the official Eclipse documentation at the official Eclipse documentation site an have a closer look into the book "Workbench User Guide".
