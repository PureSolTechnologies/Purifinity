Installation

After the requirements are met, Purifinity is installed in the following way:

  1. Purifinity is shipped for each Operating System in a ZIP file. This is to be copied onto the target computer.
  2. Simply unpack the ZIP file into a location of you choice.
  3. Start Purifinity by starting the executable file in the purifinity directory which is called purifinity or purifinity.exe.

After the installation, the client can be started. Purifinity client is already ready to use after unpacking. It can be proceeded with the System Configuration section to setup the Purifinity system.
