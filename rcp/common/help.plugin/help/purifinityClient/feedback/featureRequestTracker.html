Feature Request Tracker - Write ideas how to improve Purifinity

The Feature Request Tracker can be found at http://bugs.puresol-technologies.net.

A Feature Request Tracker is used by users and developers of a software product to request new functionality and to suggest improvements in the product. The features (as well as bugs) are reported in so-called tickets. Each ticket contains at least title and a description.

The usage of the Feature Request Tracker is exactly the same as the Bug Tracker. Only the type of the ticket needs to be set to 'Feature'. For more information, please have a look into the usage hints of the Bug Tracker.
