About

The name Purifinity is a combination of the words 'pure' or 'puritiy' and 'infinity'.

The goal is to create a source code analysis tool and platform to assure quality in the source code. For that purpose difference aspects need to be monitored:

  • Architecture
  • Design
  • Coding Style
  • Defects
  • Metrics

Purifinity is meant to be a platform to collect and present all of these aspects in one single tool to help developers, quality assurance staff, testers and managers to track the software quality on source level.

In times of polyglot programming, Purifinity needs to be language independent.

Last, but not least, Purifinity is designed to handle a lot of large projects over a long time period.
