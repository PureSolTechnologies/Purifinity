Basic Cost Construction Model (Basic COCOMO)

The Basic Cost Construction Model is one model to estimate the total cost of development for software product. The estimation is based on empirical data of different software projects. The only input into the model is the Source Line Of Code Metric (SLOC) and a rough rating about the project complexity.

Model Equations

The formulae in overview:

Sign Unit Equation Description
s Number of source lines
a Average annual developer salary
sk k sk = s / 1000.0 Number of source lines in thousands
tpm months tpm = c1 * skc2 Person Month
tpy years tpy = tpm / 12.0 Person Years
tsm month tsm = 2.5 * tpmc3 Scheduled Month
tsy years tsy = tsm / 12 Scheduled Years
p p = tpm / tsm Team Size
c currency c = tpy * a * 2.4 Estimated cost of development

The related constants matrix:

Complexity C1 C2 C3
Low 2.40 1.05 0.38
Medium 3.00 1.12 0.35
High 3.60 1.20 0.32

Discussion

This model gives only a rough estimate about the project costs. It does not take into account that source code is altered, that some technical constraints and difficulties may be cost drivers and the technical resources and developer skills may influence speed and quality. A more sophisticated model for project costs is the so called Intermediate COCOMO model.
