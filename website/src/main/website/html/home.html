Introduction

Purifinity is a source code analysis tool and platform for large and detailed analyses and visualization of architecture, design and code quality.

Features

  • Language independent analyzes and evaluation architecture
  • Plugin architecture for extensive extensibility
  • (Almost) unlimited storage space due to usage of Hadoop and HBase
  • Graph storage with DuctileDB for dependency analyses, coupling metrics and much more
  • Modern responsive Web UI based on Angular and Bootstrap

For details have a look to Features.
